# face-recognition-app

Face recognition app project completed as part of the Zero to Mastery Complete Web Developer Bootcamp.

## Initial Setup

npm install

## API Key

This app requires a [Hugging Face](https://huggingface.co/) API key to function.

Assign to an environment variable called VITE_HUGGING_FACE_API_KEY for it to be detected.

## Run demo

npm dev
